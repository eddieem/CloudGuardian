Help

Here are some common questions and troubleshooting tips to help you use the Cloud Security Configuration Analyzer:

  • How do I upload a JSON file? Click the "Choose File" button on the home page, select your JSON configuration file, and click "Upload."
  • What file format is supported? The application currently supports JSON format for configuration files.
  • What should I do if I encounter an error? If you encounter an error, make sure your JSON file is correctly formatted. If the problem persists, contact support.
